Which Scheme Is Right for You?
👧
SSY — 8.2% Tax-Free
Best rate for girl child. Deposit up to ₹1.5L/yr. Matures when girl turns 21. Withdrawals allowed at 18 for education. Fully tax-free under 80C.
🏦
PPF — 7.1% Tax-Free
Best long-term safe bet. Fully tax-free (EEE status). Deposit ₹500–₹1.5L/yr. Lock-in 15yr but extends in 5yr blocks. Loan available after 3yr.
📜
NSC — 7.7%
5-year lock. Good FD alternative. Interest is taxable but reinvested interest qualifies for 80C deduction. Collateral for bank loans accepted.
📬
MIS — 7.4% Monthly
Monthly income payout. Max ₹9L single / ₹15L joint. Good for supplementing income. Interest taxable. Can reinvest payouts into RD.
📮
TD 5yr — 7.5%
Like a bank FD but at post office. Quarterly compounding. 5-yr TD qualifies for 80C deduction. Premature withdrawal allowed after 6 months.
🎯
SCSS — 8.2% Quarterly
Highest safe rate but for 60+ only. Quarterly interest payout. Max ₹30L. 5yr term, extendable 3yr. Best for retired parents.

🏦 EPF / VPF
• Employee 12% + Employer 12% of basic
• Rate: 8.25% (FY 2024-25), revised annually
• EEE — fully tax-free (exempt-exempt-exempt)
• Withdrawal: after 58 or 5yr+ service
• VPF: voluntary extra contribution at same rate
• 80C benefit up to ₹1.5L combined
📊 NPS
• Market-linked pension scheme (equity/debt/govt)
• Extra ₹50K deduction under 80CCD(1B)
• Lock-in till 60 (Tier-I), flexible (Tier-II)
• At 60: 60% lumpsum tax-free, 40% annuity
• Returns depend on asset allocation choice
• Low expense ratio (~0.01% fund mgmt charge)

How to get a Google OAuth Client ID
1. Go to console.cloud.google.com
2. Create a new project (e.g. "FinFolio")
3. Go to APIs & Services → Library → search "Gmail API" → Enable
4. Go to APIs & Services → OAuth consent screen → External → fill in app name & email → add gmail.readonly scope → add yourself as test user
5. Go to Credentials → Create Credentials → OAuth Client ID
6. Type: Web application
7. Authorized JavaScript origins: add your app URL (e.g. https://your-app.fly.dev)
8. Copy the Client ID and paste it above
✓ Gmail API is free — no billing or credit card needed.
